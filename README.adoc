= SQL-Insight


== 正在开发中.. 作者很懒 最近在把java代码移植到kotlin

==  SQL-Insight是什么？
这个项目可以让你直观了解一条SQL背后发生了什么，
是学习和步入关系型数据库的利器。



==  SQL-Insight 是怎么运行的?
作者用Java根据MySQL原理实现了一些link:page/support.adoc[简单功能]，并用视图和动画的方式展现。

== SQL-Insight不是什么

作者没有想完整实现一个数据库的想法，只想提供一个可视化的逻辑流程。

细节上没有追求尽善尽美，但尽量追求严谨，虚心接受所有意见。
